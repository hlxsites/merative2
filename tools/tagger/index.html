Search + copy your tags

Use this page to search and choose the relevant tags for your pages, without fear of misspelling or forgetting any.

Once you click on a tag, it will populate at the bottom of the page. Re-clicking a tag will de-select it. When you have picked all the tags, click “copy” and then paste the information directly in your page's metadata (only the solid colored words will be copied, not the whole path). Use the 'clear' button if you want to start over.

  • Categories = will place your blog article in the correct category area per the main blog navigation
  • Audience = the audience group the content is created for
  • Asset type = the type of asset on the page. Primarily used for PDFs and offer registration pages
  • Page type = the type of webpage. It must be used on all web pages.
  • orgFunction = the owning organization function of the webpage. It must be used on all web pages.
  • Topics = will place your blog article in the correct topic area per the main blog navigation
  • Parent = required for analytics. It must be used on all web pages.
  • Product Family = required for analytics. It must be used on all web pages.
